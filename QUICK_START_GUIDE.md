## Step 1: Install Git (5 minutes)

1. **Download Git:**
   - Go to: **https://git-scm.com/download/win**
   - Click the download button (it will auto-detect Windows)
   - The file will be something like `Git-2.xx.x-64-bit.exe`

2. **Install Git:**
   - Run the downloaded installer
   - **Keep all default options** (just click "Next" through everything)
   - When it asks about "Default editor", choose "Nano" or "Notepad" (simpler)
   - Click "Install"
   - Wait for installation to finish

3. **Verify Installation:**
   - Close and reopen your terminal/PowerShell
   - Run: `git --version`
   - You should see something like: `git version 2.xx.x`
